Hello, Message server

Features

  1. Post推送到订阅者提供的url地址, 订阅者无须长连接或是阻塞从服务器获取消息，可用于异地网络
  2. 消息队列使用redis存储，保证消息不丢失，安全送达客户端，当服务中断或重启，未发送的消息自动发送
  3. 当消息因网络问题或其它问题不可到达，延时5次重新尝试发送，直到存储到失败队列进行人工处理
  4. 可设置用于发送消息的进程数量
  5. 足够高效，在vps测试，可达2000 qps, 满足大部分应用系统
